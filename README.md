## 🧪 ZK Proof Model (High‑Level)

Each module follows the same pattern:

- **Commitment**: hash(raw_data + salt)  
- **Proof**: zk‑SNARK or zk‑STARK verifying a statement  
- **Metadata**: non‑sensitive descriptors  
- **Verification**: on‑chain or off‑chain  
- **Revocation**: versioned commitments  
- **Delegation**: role‑based permissions
